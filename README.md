**Lite SW** is a collection of ultra-lightweight, single-file, zero-dependency web applications designed for personal productivity. Everything runs 100% locally in your browser using `localStorage`. No servers, no tracking, no databases to set up.

*🇺🇦 **Lite SW** — це колекція надлегких веб-додатків, що складаються з одного файлу та не мають залежностей. Усе працює 100% локально у твоєму браузері. Ніяких серверів, ніякого стеження.*

---

## 🛠️ Applications / Додатки

### 1. 📘 LiteFi (Personal Finance)
A robust personal finance manager in a single HTML file.
* **Multi-currency accounts:** Manage cash, cards, and debts.
* **Double-entry bookkeeping:** Seamlessly transfer funds between accounts.
* **Split Transactions:** Divide a single receipt into multiple categories.
* **Budgeting & Planning:** Schedule future payments and subscriptions.

### 2. 📦 LiteWH (Warehouse Management System)
A micro-WMS built for makers, engineers, and DIY enthusiasts.
* **Hierarchical Categories:** E.g., `Electronics ➔ Microcontrollers`.
* **Serial Number Tracking:** Maintain a pool of serials for each item.
* **BOM & Assembly Module:** Build final products (e.g., Drones) by automatically consuming raw components from the stock.
* **Smart Dispatch:** Select specific serial numbers when taking items out of stock.

## 🚀 How to Use / Як користуватися

**Zero Installation required! / Встановлення не потрібне!**

1. Download the desired HTML file (`litefi.html` or `litewh.html`).
2. Open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Start using it! Add it to your bookmarks or save it to your mobile home screen for app-like experience.

*🇺🇦 Просто завантаж HTML-файл та відкрий його у будь-якому браузері. Можеш додати його на головний екран смартфона для роботи у повноекранному режимі.*

---

## 🔐 Data Privacy / Конфіденційність
**Your data is yours.** These apps do not make any network requests. All data is saved strictly in your browser's `localStorage`. You can export your database as a `.json` file at any time for backup or syncing across devices.

*🇺🇦 **Твої дані належать тобі.** Додатки не використовують інтернет. Уся база зберігається локально. Ти можеш у будь-який момент експортувати дані у `.json` файл.*
